About this app

This app demonstrates a subset of the features offered by the Backbase Mobile SDK. It can be used in different ways. For example as an informative app about the (current) features of the Mobile SDK as well as to understand how they can build apps based on the Mobile SDK as the source code of the app is available as open source. All functionality is built using widgets. These widgets are directly built on top of the Launchpad architecture.

About Backbase

Backbase is a software company, founded in the Netherlands in 2003, which has offices in Amsterdam, the Netherlands, Atlanta and New York, United States, and London, United Kingdom. We are on a mission to help financial institutions create, manage, and optimize secure omni-channel customer interactions.
