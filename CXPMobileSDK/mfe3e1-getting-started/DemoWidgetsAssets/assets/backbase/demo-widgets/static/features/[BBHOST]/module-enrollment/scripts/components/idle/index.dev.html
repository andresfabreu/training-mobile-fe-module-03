Usage example

Assuming that the following code is defined in controller:

// Get IDLE tracker instance ctrl.idle = idleTracker({ intervalMin: 5 // default is 1 min }); // Start IDLE tracker ctrl.idle.init(); // IDLE COUNTDOWN event lpCoreBus.subscribe('lp:idle:notification', function (remainSeconds) { ctrl.idleLeftSeconds = remainSeconds; }); // Handle IDLE MAIN event lpCoreBus.subscribe('lp:idle:reaction', function () { // do something });
