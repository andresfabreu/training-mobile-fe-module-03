CXPRendererFactory Class Reference

Inherits from NSObject
Declared in CXPRendererFactory.h

Overview

Renderer factory. This factory will retrieve or create an appropiated Renderer object based on the Renderable item information. Furthermore, if preload or retaining are specified this factory will take care of caching the instance or retrieving it from the cache on subsequent calls.
