SimpleStorageComponent Class Reference

Inherits from NSObject
Declared in SimpleStorageComponent.h

Overview

Provides persistent storage capabilities based on a key-value storage. Only supports keys as strings and values as string. The information is stored as-is.

– setItem:forKey:

Stores the given value under the given key.

- (void)setItem:(NSString *)value forKey:(NSString *)key

Parameters

value

String value to store
key

String key to identify the value.

Discussion

Stores the given value under the given key.

For optimization purposes, storing a nil-value is equivalent to remove it from the storage.
