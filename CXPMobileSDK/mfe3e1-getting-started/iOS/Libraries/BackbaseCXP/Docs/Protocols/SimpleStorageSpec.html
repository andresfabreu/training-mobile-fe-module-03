– setItem::: required method

Sets a value in the given key. If the key doesn’t exists the value is inserted. If the key already exists it’s override by the new value.

- (void)setItem:(NSString *)callbackId :

Parameters

callbackId

The callbackId that this invocation refers to.
key

The key’s name to be stored
value

The value to insert / replace in the given key.

Discussion

Sets a value in the given key. If the key doesn’t exists the value is inserted. If the key already exists it’s override by the new value.

This method’s signature differs from the objective-c standard (setItem:forKey):

  1. to compliant with W3C specification of the Storage interface.
  2. to provide the correct method name in javascript plugin.setItem('key', 'value');
