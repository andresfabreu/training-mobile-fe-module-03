Renderer Protocol Reference

Conforms to NSObject
Declared in Renderer.h

Overview

Renderer protocol. Objects conforming this protocol are able to render Renderable items and control their graphical properties.

– start:error: required method

Starts the rendering of a renderable item in a given container.

- (BOOL)start:(UIView *)container error:(NSError **)error

Parameters

container

Container where the item should be rendered.
error

If an error occurs, upon return contains an NSError object that describes the problem.

Discussion

Starts the rendering of a renderable item in a given container.
