Accounts (lib/accounts)

lp-format-amount (directive)

{{account.balance}} is formatted to

Given an account, the lp-format-amount will displayed a formatted amount of money based on it's current balance.

Settings

  • lp-format-amount: The account object.

lp-accounts-select (directive)

Given a list of accounts, the lp-accounts-select will display a dropdown selection of them.

Settings

  • ng-model: The current selected account.
  • lp-accounts: Array of account objects.
