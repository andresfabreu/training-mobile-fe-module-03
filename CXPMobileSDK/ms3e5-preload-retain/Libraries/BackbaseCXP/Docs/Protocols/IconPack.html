IconPack Protocol Reference

Conforms to NSObject
Declared in IconPack.h

Overview

IconPack protocol. Objects conforming this protocol know how to generate or retrieve the different icon states. Implementations of this protocol could read files or generate UIImage’s on the fly for different states.
